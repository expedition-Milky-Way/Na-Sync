# 简介

NAS上云基于凝聚星河开发。是一个网页应用。提供了“文件监听”以及”定时备份“两种备份方式。将服务器上的文件夹以压缩包的方式上传到百度网盘。

# 如何使用

1. 在<a href="#">百度网盘开放平台</a>中申请加入并创建应用
2. 在您的nas上配置Java运行环境，并下载该应用
3. 使用和nas同一网段下的电脑浏览器，输入ip+:8828端口进入主界面
4. 设置相对应的设置并验证，就可以开始同步备份了
